    2019网文届最强作《诡秘之主》 #

    2020-05-27 星期三

    summary: 集故事性，爽，思考与复杂世界观于一体的优秀作品，截至目前我读过的最好的网文！

    最近重拾了弃读三次的《诡秘之主》，发现真的很好看，远远超出了我的预计。虽然是起点上日更的网文，但是描写细致，人物丰满，世界观浩大，三观端正！而且不管是那一条，都远远超出了普通网文，达到了出版物以上的水准。

    细节 #

    描写方面，作者对场景、建筑、衣着等没有敷衍了事，各种古代文明的元素，以及近现代西方风格都写得很细致。甚至连金融体系都非常完整，主角进账多少、出账多少，都有详细的计算。上流社会人们的生活、底层社会人们的生活，教会信仰，都信手拈来，代入感很强。

    人物塑造 #

    书中人物众多，而且各不相同，很多配角都有血有肉，他们的生活很是让人唏嘘不已。最让人服气的是，就算是微不足道的小人物，没有任何特异，甚至很贫穷——比如老科特，丽芙母女，他们只是普通工人，但却让人真切感受到他们在努力生存，他们在爱着他们身边的人，最后他们的死亡也让人不舍和震动，真切希望有神灵能将他们拯救。

    世界观 #

    世界观层层叠叠，抽丝剥茧，脑洞逆天。逻辑也能自洽。比如说“序列唯一性”设定，直接奠定了很多矛盾的基础，解决了“打什么””怎么打“的问题，不像很多小说，让人很困惑为什么老有人找主角麻烦。

    三观 #

    这本书三观端正地令人惊奇——不禁让人感慨网文三观的不正之风究竟是多么凛冽，才让端正的三观如此珍贵。主人公不惹是生非，不沾花惹草，没有女角脑残一样爱上主人公。女性角色得到应有的尊重，非常出彩。另外，主人公从来不滥杀无辜，但也没有随意路见不平拔刀相助。很喜欢和官方势力合作，能不自己动手就不自己动手（人送绰号怂神）。对于弱小者，主角团队都是能帮则帮，没有能力就想办法；对于邪恶者也不吝惩戒。

    雷 #

    本书毒点在于，就是到最后布局太大，战斗太多，看着啰嗦，且有开挂之嫌。这些毒点相对于其他小说的毒点，只能说真的不算什么毒点了。

    最后强烈推荐 #

    是一本名副其实的现象级好书啊！
